### 模块4: 文档生成模块

- [ ] US-015: 保存图表源码 - Mermaid 源码保存到独立文件

### 模块5: 配置管理模块
- [ ] US-016: 生成配置文件 - 自动创建 .claude/settings.json
- [ ] US-017: 生成.gitignore - 自动创建适合项目的 .gitignore

## 用户旅程地图

### 旅程1: 新手首次使用
```
安装CLI工具 → 运行 ai-dev init → 选择项目类型 → 输入项目信息
→ 项目创建成功 → 打开项目 → 使用Claude Code → 输入"我想做..."
→ 进入需求分析流程 → 完成需求分析 → 生成PRD → 开始开发
```

### 旅程2: 老手快速启动
```
npx @ai-dev-template/cli init → 选择模板 → Enter → 打开项目
→ 已有清晰需求 → 直接"开始架构设计" → 开始开发
```

### 旅程3: 团队协作使用
```
项目负责人: 运行需求分析 → 生成PRD → 分享PRD给团队
→ 架构师: 基于PRD开始架构设计
→ 开发者: 基于架构开始编码
→ 测试: 基于验收标准编写测试
```
